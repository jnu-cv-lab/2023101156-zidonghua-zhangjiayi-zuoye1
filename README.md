# 图像处理基础实验 (OpenCV)
## 功能特点
· 图像读取：通过文件对话框选择图片，支持常见格式（jpg、jpeg、png）。
· 信息输出：显示图像的宽度、高度、通道数及像素数据类型。
· 图像显示：原图、灰度图、修改后的图像均自动缩放至合适窗口大小，避免窗口过大。
· 灰度转换：将彩色图像转换为灰度图并保存。
· 区域裁剪：从图像左上角裁剪 50×50 像素区域并保存。
· 像素修改：将原图左上角 50×50 区域置为黑色并保存。

## 开发环境
· 操作系统：Windows 11 + WSL (Ubuntu 22.04) 或原生 Linux / macOS
· 开发工具：Visual Studio Code（推荐）
· 编程语言：Python 3.10 或更高版本
· 依赖库：OpenCV、NumPy、Tkinter

## 安装步骤
1. 将本项目克隆到本地。
2. 建议创建 Python 虚拟环境（可选）。
3. 安装 OpenCV 和 NumPy 库。若使用 pip，可通过常规方式安装。
4. 若在 WSL 环境下运行，请确保图形界面正常显示（Windows 11 的 WSLg 已集成支持，Windows 10 需额外配置 X Server）。

## 使用方法
1. 在终端中进入项目目录，运行主程序。
2. 程序启动后会弹出文件选择对话框，请选择一张测试图片。
3. 程序依次显示：
   · 缩放后的原始图像
   · 缩放后的灰度图像
   · 裁剪出的 50×50 区域（原始大小）
   · 左上角涂黑后的修改图像（缩放显示）
4. 每张图像显示时，按任意键关闭当前窗口并继续。
5. 控制台会输出图像的基本信息以及各步骤的保存状态。
6. 程序运行结束后，当前目录下会生成三个文件：
   · 灰度图（gray_image.jpg）
   · 裁剪区域（cropped_region.jpg）
   · 左上角涂黑后的图像（modified_image.jpg）

## 注意事项
· 若未选择图片，程序会提示错误并退出。
· 确保运行目录具有写入权限，否则无法保存输出文件。
· 在 WSL 环境下若图像窗口无法显示，请检查 X Server 配置或改用 WSLg（Windows 11）。
· 若出现 cv2.imshow 无响应，可考虑安装 opencv-python-headless 并使用其他显示方式（如 matplotlib）作为替代。
